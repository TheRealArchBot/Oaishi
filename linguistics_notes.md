# 1. Ferdinand de Saussure

> 🎓 **Who is he?** Swiss linguist (1857–1913). Called the **"Father of Modern Linguistics"**. His lectures were compiled into the book *Course in General Linguistics* (1916) after his death by his students.

## Saussure's Core Ideas

### A) Language as a SYSTEM
Saussure said language is not just a list of words — it is a **system of signs** where everything gets meaning by being *different from* other things.

> **Example:** "Hot" only means something because "cold" exists. Without contrast, there's no meaning.

---

### B) The Linguistic Sign

Saussure broke every word into **two inseparable parts** — like two sides of a paper. You can never have one without the other.

```
┌─────────────────────────────────────────┐
│           LINGUISTIC SIGN               │
│                                         │
│   ┌──────────────┬──────────────────┐   │
│   │  SIGNIFIER   │    SIGNIFIED     │   │
│   │              │                  │   │
│   │ Sound-Image  │   Mental Concept │   │
│   │  (the word   │  (the idea in    │   │
│   │  "tree" /triː/)│  your head 🌳)  │   │
│   └──────────────┴──────────────────┘   │
│                                         │
│   Together = one SIGN                   │
└─────────────────────────────────────────┘
```

| Term | What it means | Example |
|------|--------------|---------|
| **Signifier** | The sound or written form of a word | The sounds /t/ /r/ /iː/ |
| **Signified** | The mental concept / idea triggered | Your brain's image of a tree 🌳 |
| **Sign** | Both together = a complete linguistic unit | The word "tree" as we use it |

> 🔥 **Key Point for 10-mark:** The relationship between signifier and signified is **ARBITRARY** — meaning there is NO natural connection. The word "dog" doesn't sound like a dog. Different languages use totally different sounds for the same animal (chien in French, Hund in German). This is Saussure's **Arbitrariness of the Sign**.

---

### C) Langue vs Parole

```
     LANGUAGE (Abstract System)
     ──────────────────────────
           │
     ┌─────┴─────┐
     │           │
  LANGUE      PAROLE
  (Rules)   (Actual speech)
     │           │
  Grammar     What you say
  system       today
     │           │
  Shared by   Unique to each
  everyone    person/moment
```

| Feature | Langue | Parole |
|---------|--------|--------|
| **What is it?** | Abstract system of language rules | Actual, real speech acts |
| **Who has it?** | Shared by the entire speech community | Individual speakers |
| **Changes?** | Slow, collective change | Varies per person, per moment |
| **Example** | The rules of English grammar | You saying "gonna" instead of "going to" |
| **Analogy** | Recipe book 📖 | Your actual cooking attempt 🍳 |

> 🧠 **Memory Trick:** Langue = **L**aw of language. Parole = **P**ersonal performance.

---

### D) Synchronic vs Diachronic Linguistics

| Type | Meaning | Example |
|------|---------|---------|
| **Synchronic** | Studying language at ONE point in time | English grammar TODAY |
| **Diachronic** | Studying language CHANGE over time (historical) | How "thou" disappeared from English |

> Saussure argued linguistics should focus more on **synchronic** study — understand the system as it exists NOW, not just its history.

---

### E) Syntagmatic vs Paradigmatic Relations

```
Syntagmatic (horizontal — left to right):
  The  →  cat  →  sat  →  on  →  the  →  mat

Paradigmatic (vertical — swap options):
         cat
         dog       ← any of these can replace "cat"
         bird
         child
```

- **Syntagmatic**: How words combine in sequence (sentence structure)
- **Paradigmatic**: The pool of words you *could* choose at each position

# 2. Noam Chomsky

> 🎓 **Who is he?** American linguist (born 1928). Revolutionized linguistics in the 1950s–60s. His most famous idea: **humans are born with language ability already wired in**.

## Chomsky's Core Ideas

### A) Generative Grammar / Transformational Grammar
Chomsky argued that a finite set of grammar rules can **generate infinite sentences**. You've never heard most sentences before, yet you understand them instantly.

> "Colorless green ideas sleep furiously." — Chomsky's famous example
> ✅ Grammatically correct | ❌ Completely meaningless
> This proves grammar (syntax) and meaning (semantics) are SEPARATE systems.

---

### B) Competence vs Performance

```
┌────────────────────────────────────────────┐
│                                            │
│  COMPETENCE                                │
│  ─────────                                 │
│  Your internalized, unconscious            │
│  knowledge of grammar rules.               │
│  You "know" English grammar even if        │
│  you can't explain it.                     │
│                                            │
│  PERFORMANCE                               │
│  ───────────                               │
│  What you actually say/write.              │
│  Full of mistakes, hesitations,            │
│  "um"s and "uh"s.                          │
│                                            │
└────────────────────────────────────────────┘
```

| | Competence | Performance |
|--|-----------|------------|
| **Type** | Abstract mental knowledge | Actual language use |
| **Errors?** | No — it's perfect in theory | Yes — slips, typos, nerves |
| **Example** | Knowing "subject comes before verb" | Saying "He go school" when nervous |
| **Analogy** | Knowing all chess rules perfectly | Playing a real game with blunders |

> Compare with Saussure: Competence ≈ Langue / Performance ≈ Parole

---

### C) Universal Grammar (UG)
Chomsky claimed all humans are born with a **Language Acquisition Device (LAD)** — a mental module pre-loaded with the basic structure of all possible human languages.

```
      Baby born
          │
          ▼
   ┌─────────────┐
   │     LAD     │   ← Language Acquisition Device
   │ (built-in   │     (Universal Grammar)
   │  grammar    │
   │  template)  │
   └─────────────┘
          │
     Exposed to
     any language
          │
          ▼
   Learns that language
   naturally by age 5
```

> **Evidence:** Children make the SAME type of errors across all languages. They overgeneralize rules (saying "goed" instead of "went") — showing they're applying a rule, not just memorizing.

---

### D) Surface Structure vs Deep Structure

```
Same deep structure → Different surface structures:
──────────────────────────────────────────────────
Deep:    [Someone] opened the door.

Surface 1: The boy opened the door.
Surface 2: The door was opened by the boy.   (passive)

Different deep structures → Same surface structure:
───────────────────────────────────────────────────
Surface:  "I saw the man with the telescope."

Deep 1: I [used a telescope] to see the man.
Deep 2: I saw [the man who had a telescope].
```

| Structure | Definition |
|-----------|-----------|
| **Deep Structure** | The underlying logical meaning of a sentence |
| **Surface Structure** | The actual words/order you see/hear |

## The International Phonetic Alphabet (IPA)

The IPA gives **one symbol = one sound**, always. Used worldwide.

- Phone symbols go in **square brackets [ ]**
- Phoneme symbols go in **slash brackets / /**

## Places of Articulation (Where sounds are made)

```
CROSS-SECTION OF THE HUMAN VOCAL TRACT:

           Nasal cavity
              ↑
    ┌─────────────────────────┐
    │   ↑Velum(soft palate)   │
    │  Palate──────┐          │
    │  Alveolar    │          │
    │  ridge──┐    │          │
    │  Teeth──┤    │   Throat │
    │  Lips───┤    │          │
    └─────────────────────────┘
              ↓
           Larynx
          (Voice Box)
```

| Place | Organs Involved | Example Sounds |
|-------|----------------|----------------|
| **Bilabial** | Both lips together | /p/, /b/, /m/ |
| **Labiodental** | Lower lip + upper teeth | /f/, /v/ |
| **Dental** | Tongue tip + upper teeth | /θ/ (think), /ð/ (this) |
| **Alveolar** | Tongue tip + alveolar ridge (behind top teeth) | /t/, /d/, /s/, /z/, /n/, /l/ |
| **Post-alveolar** | Tongue just behind alveolar ridge | /ʃ/ (shop), /ʒ/ (measure) |
| **Palatal** | Front of tongue + hard palate | /j/ (yes) |
| **Velar** | Back of tongue + soft palate (velum) | /k/, /g/, /ŋ/ (sing) |
| **Glottal** | At the larynx/glottis | /h/, /ʔ/ (uh-oh) |

---

## Manners of Articulation (How sounds are made)

| Manner | Description | Examples |
|--------|-------------|---------|
| **Plosive/Stop** | Complete airflow blockage then release | /p/, /b/, /t/, /d/, /k/, /g/ |
| **Fricative** | Partial blockage → turbulent air (hissing) | /f/, /v/, /s/, /z/, /ʃ/, /θ/ |
| **Affricate** | Stop + fricative combined | /tʃ/ (church), /dʒ/ (judge) |
| **Nasal** | Air through nose, mouth blocked | /m/, /n/, /ŋ/ |
| **Lateral** | Air flows around sides of tongue | /l/ |
| **Approximant** | Articulators approach but don't block | /w/, /j/, /r/ |

## Vowel Sounds

Vowels are made with an OPEN vocal tract. Described by:
1. **Tongue height** (high, mid, low)
2. **Tongue backness** (front, central, back)
3. **Lip rounding** (rounded / unrounded)

### The Vowel Quadrilateral Diagram

```
              FRONT    CENTRAL    BACK
              ─────    ───────    ────
HIGH (CLOSE)   i·y       ɨ·ʉ      ɯ·u
               │                   │
MID-HIGH       e·ø       ə·ɵ      ɤ·o
               │         │         │
MID-LOW        ɛ·œ       ɐ        ʌ·ɔ
               │                   │
LOW (OPEN)     a·æ                 ɑ·ɒ

(left symbol = unrounded, right symbol = rounded)
```

### Common English Vowels

| Sound | Example word | Description |
|-------|-------------|-------------|
| /iː/ | f**ee**t | High, front, unrounded (long) |
| /ɪ/ | f**i**t | High, front, unrounded (short) |
| /e/ | s**a**y | Mid, front, unrounded |
| /æ/ | c**a**t | Low, front, unrounded |
| /ɑː/ | c**ar** | Low, back, unrounded (long) |
| /ɒ/ | h**o**t | Low, back, rounded |
| /ɔː/ | l**aw** | Mid, back, rounded (long) |
| /ʊ/ | b**oo**k | High, back, rounded (short) |
| /uː/ | m**oo**n | High, back, rounded (long) |
| /ʌ/ | b**u**s | Mid, central, unrounded |
| /ə/ | **a**bout | Mid, central (schwa — most common English vowel!) |
| /ɜː/ | b**ir**d | Mid, central (long) |

## Diphthongs

> A **diphthong** is a vowel that **glides** from one position to another within a single syllable. Think of it as a *moving vowel*.

```
Diphthong = vowel 1 → slides to → vowel 2
                (within ONE syllable)
```

| Diphthong | Example | Movement |
|-----------|---------|---------|
| /eɪ/ | **day**, **say** | mid-front → high-front |
| /aɪ/ | **my**, **high** | low → high-front |
| /ɔɪ/ | **boy**, **coin** | back-round → high-front |
| /aʊ/ | **now**, **cow** | low → high-back |
| /əʊ/ | **go**, **home** | central → back |
| /ɪə/ | **here**, **ear** | high-front → central |
| /eə/ | **air**, **bear** | mid-front → central |
| /ʊə/ | **tour**, **cure** | high-back → central |

> 🧠 **Memory tip:** Diphthongs are "two-vowel journeys" in one syllable.

# 4. Phonology

> **Definition:** Phonology studies how sounds **function and pattern** within a particular language. It's about the **rules** governing sound systems.

## Key Concepts

### Phone vs Phoneme vs Allophone

```
┌────────────────────────────────────────────────────────┐
│                                                        │
│  PHONE [ ]                                             │
│  Any single speech sound, regardless of meaning.      │
│  Just a raw physical sound.                            │
│  Example: [pʰ] in "pin" and [p] in "spin" are         │
│  two different phones.                                 │
│                                                        │
│  PHONEME / /                                           │
│  A sound that CHANGES MEANING in a language.           │
│  /p/ vs /b/ in "pin" vs "bin" → different meaning     │
│  → they are phonemes (minimal pair test!)              │
│                                                        │
│  ALLOPHONE                                             │
│  Different versions of the SAME phoneme that do NOT   │
│  change meaning — just context-dependent variations.  │
│  [pʰ] in "pin" and [p] in "spin" are allophones       │
│  of the phoneme /p/. Swap them → still same meaning.  │
│                                                        │
└────────────────────────────────────────────────────────┘
```

### Minimal Pairs
Two words that differ by **exactly ONE sound** and have different meanings.

| Pair | Sounds Differ | Proves these are phonemes |
|------|--------------|--------------------------|
| pin / bin | /p/ vs /b/ | /p/ and /b/ are phonemes |
| cat / bat | /k/ vs /b/ | /k/ and /b/ are phonemes |
| seat / feet | /s/ vs /f/ | /s/ and /f/ are phonemes |
| ship / chip | /ʃ/ vs /tʃ/ | both are phonemes |

---

### Phonological Processes (how sounds change in context)

| Process | Definition | Example |
|---------|-----------|---------|
| **Assimilation** | Sound becomes similar to neighboring sound | "in possible" → "impossible" (/n/ → /m/ before /p/) |
| **Elision** | Sound dropped in fast speech | "chocolate" → "choc-late" |
| **Epenthesis** | Extra sound inserted | "athlete" pronounced "ath-e-lete" |
| **Vowel reduction** | Unstressed vowels become /ə/ | "photograph" → "photography" (second vowel shifts) |

# 5. Morphology

> **Definition:** Morphology is the study of the **internal structure of words** — how words are built from smaller meaningful units.

## Key Units

```
MORPHOLOGY BUILDING BLOCKS:
─────────────────────────────────────────────

MORPH
─────
A physical, observable form.
Example: the letters/sounds you see in "cats" = c-a-t-s

MORPHEME
────────
The SMALLEST unit of MEANING.
Cannot be broken down further.
"cats" has TWO morphemes: [cat] + [s]

ALLOMORPH
─────────
Different physical forms of the SAME morpheme.
The plural morpheme has 3 forms:
/s/ in cats  |  /z/ in dogs  |  /ɪz/ in buses
All mean "more than one" → all allomorphs of {PLURAL}
```

---

## Classification of Morphemes

```
                    MORPHEMES
                        │
          ┌─────────────┴─────────────┐
          │                           │
       FREE                        BOUND
    MORPHEMES                    MORPHEMES
    (stand alone)               (need a base)
          │                           │
    ┌─────┴─────┐             ┌───────┴───────┐
    │           │             │               │
LEXICAL     FUNCTIONAL     DERIVATIONAL   INFLECTIONAL
  (content)  (grammar)      (new words)    (grammar)
    │           │             │               │
  cat, run,   the, and,    un-, -ness,     -s, -ed,
  happy, big  of, to       re-, -ful       -ing, -er
```

### Free Morphemes (can stand alone)

| Type | Definition | Examples |
|------|-----------|---------|
| **Lexical/Content** | Carries main meaning | cat, run, happy, blue |
| **Functional/Grammar** | Grammatical role, little meaning alone | the, a, of, and, to |

### Bound Morphemes (must attach to a base)

| Type | What it does | Examples |
|------|-------------|---------|
| **Derivational** | Creates a NEW word or changes word class | un+happy = unhappy (changes meaning); happy → happiness (changes class: adj→noun) |
| **Inflectional** | Only changes grammatical info, NOT a new word | dogs (plural), walked (past), bigger (comparative) |

---

## The 8 Inflectional Suffixes in English

> ⚠️ **EXAM MUST-KNOW — Memorize all 8:**

| Suffix | Function | Example |
|--------|---------|---------|
| **-s / -es** | Noun plural | cats, buses |
| **-'s** | Noun possessive | cat's |
| **-s** | 3rd person singular verb | he run**s** |
| **-ed** | Past tense | walk**ed** |
| **-en / -ed** | Past participle | broken, walked |
| **-ing** | Present participle | walk**ing** |
| **-er** | Comparative adjective | tall**er** |
| **-est** | Superlative adjective | tall**est** |

---

## Derivational Affixes

| Affix | Type | Function | Example |
|-------|------|---------|---------|
| **un-** | Prefix | Negation | unhappy |
| **re-** | Prefix | Again | rewrite |
| **pre-** | Prefix | Before | preview |
| **-ness** | Suffix | Adj → Noun | happiness |
| **-ful** | Suffix | Noun → Adj | beautiful |
| **-less** | Suffix | Without | hopeless |
| **-tion** | Suffix | Verb → Noun | action |
| **-ly** | Suffix | Adj → Adverb | quickly |
| **-er** | Suffix | Verb → Noun (agent) | teacher |
| **-ize** | Suffix | Noun → Verb | modernize |

---

## Word Formation Processes

| Process | Definition | Example |
|---------|-----------|---------|
| **Affixation** | Adding prefix/suffix | un + happy = unhappy |
| **Compounding** | Two free morphemes join | tooth + paste = toothpaste |
| **Blending** | Two words merged | breakfast + lunch = **brunch** |
| **Clipping** | Word shortened | examination → **exam** |
| **Acronym** | Initials pronounced as word | NASA, NATO |
| **Initialism** | Initials pronounced separately | BBC, USA |
| **Conversion** | Word class changes, no affix | "email" (noun) → "to email" (verb) |
| **Borrowing** | Word taken from another language | café (French), kindergarten (German) |
| **Coinage** | Entirely new word invented | Google, Kleenex |
| **Backformation** | New word formed by removing supposed affix | "edit" from "editor" |

## IC Analysis (Immediate Constituent Analysis)

> **Definition:** Breaking a sentence or word into its **immediate constituents** (direct parts) step by step, like cutting a pizza smaller and smaller.

### How to do IC Analysis:

```
Sentence: "The old man sings."

Step 1: First cut (two main parts)
─────────────────────────────────
[The old man]  |  [sings]
     NP              VP

Step 2: Cut the NP further
──────────────────────────
[The old man]
  │        │
[The]   [old man]
          │     │
        [old]  [man]

TREE DIAGRAM:
                    S
                   / \
                 NP    VP
                / \     |
              Det  NP   V
              │   / \   │
             The Adj  N sings
                  │   │
                 old  man
```

### Word-level IC Analysis

```
Word: "unhappiness"

            unhappiness
                 │
          ┌──────┴──────┐
        [un]        [happiness]
                        │
                 ┌──────┴──────┐
              [happy]        [ness]
```

> 🚨 **Exam tip:** IC Analysis connects to Chomsky's **Surface Structure** — what you actually see/hear. But the same surface can have different **Deep Structures** (meaning).

### Ambiguity in IC Analysis

> One sentence can have TWO different tree structures = **structural ambiguity**

```
"I saw the man with the telescope."

TREE 1: (I used the telescope to see)
         S
        / \
       NP   VP
       │   / \
       I  V    NP
          │   / \
         saw NP  PP
             │  / \
           man P   NP
               │    │
              with telescope

TREE 2: (the man had the telescope)
         S
        / \
       NP   VP
       │   / \
       I  V    NP
          │   / \
         saw Det   N'
              │   / \
             the  N   PP
                  │  / \
                 man P   NP
                     │    │
                    with telescope
```

# 6. Syntax

> **Definition:** Syntax is the study of the **rules governing the arrangement of words into sentences** — essentially, grammar at the sentence level.

## Phrase Structure

Every sentence is built from **phrases**. A phrase is a group of words centered around one main word (the **head**).

| Phrase Type | Head | Example |
|-------------|------|---------|
| **NP** (Noun Phrase) | Noun | *the big red ball* |
| **VP** (Verb Phrase) | Verb | *runs quickly* |
| **AP** (Adjective Phrase) | Adjective | *very happy* |
| **PP** (Prepositional Phrase) | Preposition | *on the table* |
| **AdvP** (Adverb Phrase) | Adverb | *quite slowly* |

## Phrase Structure Rules (PSR)

```
S  →  NP  VP
NP →  (Det) (Adj) N (PP)
VP →  V (NP) (PP) (AdvP)
PP →  P NP
```

## Sentence Tree Diagram

```
Sentence: "The cat sat on the mat."

              S
           ┌──┴──┐
          NP     VP
        ┌──┴──┐  ┌──┴───┐
       Det    N  V      PP
        │     │  │    ┌──┴───┐
       The  cat sat   P     NP
                      │   ┌──┴──┐
                      on Det    N
                         │      │
                        the    mat
```

## Types of Sentences

| Type | Structure | Example |
|------|-----------|---------|
| **Simple** | One independent clause | The dog barked. |
| **Compound** | Two independent clauses joined by conjunction | The dog barked and the cat ran. |
| **Complex** | One main + one subordinate clause | The dog barked because it was scared. |
| **Compound-Complex** | Two independent + one subordinate | The dog barked and the cat ran because a stranger arrived. |

## Sentence Types by Purpose

| Type | Function | Example |
|------|---------|---------|
| **Declarative** | Makes a statement | The sky is blue. |
| **Interrogative** | Asks a question | Is the sky blue? |
| **Imperative** | Gives a command | Sit down! |
| **Exclamatory** | Expresses strong emotion | What a beautiful sky! |

# 7. Semantics

> **Definition:** Semantics is the study of **meaning** in language — how words, phrases, and sentences convey meaning.

## Types of Meaning

| Type | Description | Example |
|------|-------------|---------|
| **Conceptual/Denotative** | Dictionary definition, literal meaning | "dog" = four-legged domestic canine |
| **Connotative** | Associated/emotional meaning | "dog" can mean loyal friend or insult |
| **Social meaning** | What language reveals about social context | "Whom" vs "who" = formal vs casual |
| **Affective meaning** | Speaker's feelings/attitude | "It's fine." (tone = actually annoyed) |
| **Collocative meaning** | Words that typically go together | "pretty girl" but "handsome boy" |

## Lexical Relations

### Synonymy
Words with the **same or similar meaning**.
- big / large / enormous
- sad / unhappy / melancholy

> True synonyms are rare — usually there are subtle differences (register, connotation, colocation).

### Antonymy
Words with **opposite meanings**.

| Type | Description | Example |
|------|-------------|---------|
| **Gradable antonyms** | Have degrees between them | hot — warm — lukewarm — cool — cold |
| **Complementary antonyms** | No middle ground, binary | alive/dead, married/single |
| **Relational/Converse antonyms** | Each presupposes the other | buy/sell, teacher/student, above/below |

### Hyponymy (Hierarchy of meaning)

```
        ANIMAL (hypernym/superordinate)
           │
    ┌──────┼──────┐
    │      │      │
   DOG    CAT   BIRD   ← hyponyms
    │
  ┌─┴─────┐
  │       │
POODLE LABRADOR  ← co-hyponyms (at same level)
```

- **Hypernym/Superordinate**: The general category (animal)
- **Hyponym**: A specific type (dog is a hyponym of animal)
- **Co-hyponyms**: Siblings at same level (dog, cat, bird)

### Polysemy vs Homonymy

```
POLYSEMY:
─────────
One word → MULTIPLE RELATED meanings
"bank":  1. River bank (land beside water)
         2. Money bank
These evolved from the same origin — meanings are related.

HOMONYMY:
─────────
Same form → UNRELATED meanings
"bat":   1. Flying mammal 🦇
         2. Cricket/baseball bat 🏏
Totally unrelated — just happen to sound the same.
```

| Feature | Polysemy | Homonymy |
|---------|---------|---------|
| Same word? | Yes, one word | Same form, different words |
| Meanings related? | YES | NO |
| Example | "head" (body part / head of company) | "bat" (animal / sports equipment) |

### Meronymy (Part-whole relations)
- "finger" is a **meronym** of "hand"
- "wheel" is a meronym of "car"
- "page" is a meronym of "book"

---

## Semantic Roles (Thematic Roles)

> In a sentence, each noun phrase plays a **semantic role** — what role it has in the described event.

| Role | Description | Example |
|------|-------------|---------|
| **Agent** | Deliberate doer of action | **The boy** kicked the ball. |
| **Patient/Theme** | Thing affected by action | The boy kicked **the ball**. |
| **Experiencer** | Feels/perceives something | **She** loves chocolate. |
| **Instrument** | Tool used to do action | She cut the cake **with a knife**. |
| **Beneficiary** | Who benefits | He bought flowers **for her**. |
| **Location** | Where action happens | She danced **in the hall**. |
| **Source** | Where something comes from | He traveled **from London**. |
| **Goal** | Destination/endpoint | She sent the letter **to Paris**. |

---

## Semantic Relations in Sentences

| Relation | Definition | Example |
|----------|-----------|---------|
| **Entailment** | If A is true, B must be true | "John killed the spider" entails "The spider is dead" |
| **Presupposition** | Background assumption | "John stopped smoking" presupposes he smoked before |
| **Contradiction** | Two statements can't both be true | "It is raining" / "It is not raining" |
| **Anomaly** | Semantically odd/impossible | "Colorless green ideas sleep furiously" |

# 8. Pragmatics

> **Definition:** Pragmatics studies how **context** determines meaning — the gap between what is literally said and what is actually meant.

> 🔑 The key idea: humans constantly mean MORE than they say. Language depends on shared knowledge, context, and social rules.

---

## Speech Act Theory (Austin & Searle)

Philosopher J.L. Austin noticed that utterances aren't just descriptions — they are **actions**.

```
"I now pronounce you husband and wife."
→ This DOES something — it doesn't just describe!

Every utterance has THREE levels:
─────────────────────────────────

┌──────────────────────────────────────────────────┐
│  LOCUTIONARY ACT                                 │
│  The literal, face-value meaning.                │
│  "It's cold in here."  = room temperature is low │
├──────────────────────────────────────────────────┤
│  ILLOCUTIONARY ACT                               │
│  The intended action/force behind the words.     │
│  "It's cold in here." = PLEASE CLOSE THE WINDOW  │
├──────────────────────────────────────────────────┤
│  PERLOCUTIONARY ACT                              │
│  The actual effect on the listener.              │
│  Listener closes the window 🪟                   │
└──────────────────────────────────────────────────┘
```

### Types of Illocutionary Acts (Searle's Classification)

| Type | Description | Example |
|------|-------------|---------|
| **Assertives** | State a fact or belief | "The Earth is round." |
| **Directives** | Try to get listener to do something | "Pass the salt." / "Could you...?" |
| **Commissives** | Commit the speaker to something | "I promise I'll be there." |
| **Expressives** | Express emotional state | "I'm so sorry." / "Congratulations!" |
| **Declarations** | Words that change reality when spoken | "You're fired." / "I declare war." |

---

## Grice's Cooperative Principle & Maxims

Paul Grice (1975) argued that conversation works because speakers follow an unspoken **Cooperative Principle:**

> *"Make your conversational contribution as required, at the stage it occurs, by the accepted purpose or direction of the talk exchange."*

### The Four Maxims

```
┌─────────────────────────────────────────────────────┐
│                 GRICE'S MAXIMS                      │
├──────────────┬──────────────────────────────────────┤
│ QUANTITY     │ Say enough — not too much, not too   │
│              │ little.                              │
│              │ ✅ "I have two cats."                │
│              │ ❌ "I have two cats and they eat and │
│              │ sleep and breathe and..."            │
├──────────────┼──────────────────────────────────────┤
│ QUALITY      │ Say only what you believe is true.   │
│              │ Don't lie or claim without evidence. │
│              │ ✅ "It might rain later."            │
│              │ ❌ "It WILL definitely rain." (if    │
│              │ you don't actually know)             │
├──────────────┼──────────────────────────────────────┤
│ RELATION     │ Be relevant. Stay on topic.          │
│ (Relevance)  │ Q: "What time is it?"               │
│              │ ✅ "About 3pm."                      │
│              │ ❌ "Bananas are yellow." 🍌          │
├──────────────┼──────────────────────────────────────┤
│ MANNER       │ Be clear, orderly, brief, unambiguous│
│              │ ✅ "He left early."                  │
│              │ ❌ "The event of his departure       │
│              │ occurred at a non-late temporal      │
│              │ juncture."                           │
└──────────────┴──────────────────────────────────────┘
```

### Flouting the Maxims (Conversational Implicature)

> **Flouting** = deliberately breaking a maxim to imply something. The listener infers the real meaning.

| Maxim Flouted | Example | What's implied |
|--------------|---------|---------------|
| **Quality** | "Oh yeah, he's a REAL genius." (sarcastically) | He's actually stupid |
| **Quantity** | "She's okay, I guess." (when asked for reference letter) | She's not actually good |
| **Relation** | "Nice weather today." (when asked if you did homework) | I didn't do the homework |
| **Manner** | "She, uh, visited some... friends." | Something suspicious happened |

---

## Deixis (Pointing through language)

> **Deixis** = words whose meaning depends entirely on **context** (who says it, when, where).

| Type | Words | Example |
|------|-------|---------|
| **Person deixis** | I, you, we, they | "I will see you tomorrow" — means different people depending on speaker |
| **Place deixis** | here, there, this, that | "Come here" — depends on where speaker is |
| **Time deixis** | now, then, yesterday, soon | "Yesterday was fun" — depends on WHEN it's said |
| **Discourse deixis** | "as mentioned above", "the following" | Refers to other parts of the text |

---

## Politeness Theory (Brown & Levinson)

> People have a **face** — their public image/dignity. Politeness strategies protect face.

```
FACE:
──────────────────────────────────────────
POSITIVE FACE = desire to be liked, approved of
NEGATIVE FACE = desire to not be imposed on, to be free

FACE-THREATENING ACTS (FTAs):
Any speech that threatens someone's face:
- Requests → threaten negative face
- Criticism → threatens positive face
- Orders → threaten negative face

POLITENESS STRATEGIES:
─────────────────────
Bald on record:    "Give me the salt."         (direct, no softening)
Positive polite:   "You're so thoughtful — could you pass the salt?"
Negative polite:   "I'm sorry to bother you, but could you possibly...?"
Off record:        "This food really needs salt..." (hint)
Don't do FTA:      Stay silent
```

# 9. Sociolinguistics

> **Definition:** Sociolinguistics studies the relationship between **language and society** — how social factors (class, gender, age, ethnicity, region) influence language use.

---

## Language Variation

> Language is NOT uniform. Different people in different places/groups speak differently.

```
       LANGUAGE VARIATION
              │
    ┌─────────┼──────────┐
    │         │          │
REGIONAL   SOCIAL    SITUATIONAL
VARIATION VARIATION  VARIATION
    │         │          │
Dialects  Class,    Formal vs
Accents   gender,   informal
          age       register
```

### Types of Variation

| Type | Description | Example |
|------|-------------|---------|
| **Regional dialect** | Language variety based on geography | British "lift" vs American "elevator" |
| **Social dialect (sociolect)** | Based on social group | Teen slang vs elderly speech |
| **Idiolect** | Individual's unique speech style | Your own personal way of talking |
| **Register** | Variation based on situation/context | Doctor-patient talk vs friends chatting |
| **Gender variation** | Men and women speak differently | Women use more hedges ("I think", "maybe") on average |
| **Age variation** | Language changes across generations | Young people adopt new slang |
| **Ethnolect** | Language variety associated with ethnic group | African-American Vernacular English (AAVE) |

---

## Code-Switching

> Switching between two or more languages/dialects within a single conversation or even sentence.

```
Example (English/Spanish switching):
"I went to the store, y compré leche."
       (English)    (Spanish: "and bought milk")
```

- **Inter-sentential switching**: Switch between sentences
- **Intra-sentential switching**: Switch within a sentence
- **Tag-switching**: Inserting a tag from another language ("right?", "no?")

---

## Diglossia

> A situation where two varieties of a language (or two languages) exist side by side in a community, each used for different purposes.

| Variety | Used for | Example |
|---------|---------|---------|
| **High (H)** | Formal contexts: religion, education, government | Classical Arabic, Standard German |
| **Low (L)** | Everyday conversation, home, informal | Egyptian Arabic dialect, Swiss German |

---

## Language and Power

- **Dominant groups** control language norms (Standard English is the prestige variety)
- Non-standard dialects are often stigmatized, even though linguistically equal
- **Linguistic imperialism**: powerful languages (like English) suppress minority languages

---

## Language and Gender (Key Scholars)

### Robin Lakoff — "Women's Language" features:
1. More hedges ("sort of", "I think")
2. More tag questions ("It's cold, isn't it?")
3. More polite forms
4. More standard pronunciation
5. More empty adjectives ("lovely", "divine")

### Deborah Tannen — "Different but Equal":
- Men use **report talk** (information, status)
- Women use **rapport talk** (connection, feelings)
- Misunderstanding between genders = **cross-cultural communication**

---

## Context and Meaning

> In sociolinguistics, **context** determines meaning just as much as the words themselves.

```
TYPES OF CONTEXT:
─────────────────────────────────────────
Physical context:   WHERE something is said (classroom vs pub)
Social context:     WHO says it to whom (boss to employee vs friend to friend)
Temporal context:   WHEN it's said (morning greeting vs farewell)
Cultural context:   WHAT culture the speakers belong to
Linguistic context: WHAT was said before (co-text)
```

# 10. Psycholinguistics

> **Definition:** Psycholinguistics studies the **mental processes** involved in language — how we acquire, comprehend, produce, and store language in the brain.

---

## Language Acquisition

### First Language Acquisition (L1)

```
STAGES OF LANGUAGE DEVELOPMENT:
─────────────────────────────────
Birth–6 months:   Cooing, crying, biological sounds
6–12 months:      Babbling (ba-ba-ba, ma-ma-ma)
12–18 months:     First words (holophrastic stage)
                  One word = whole message
                  "Milk!" = "I want milk!"
18–24 months:     Two-word stage (telegraphic speech)
                  "Daddy go", "more juice"
2–3 years:        Multi-word sentences, rapid growth
3–5 years:        Complex sentences, full grammar
5+:               Continued expansion, literacy
```

### Key Theories of L1 Acquisition

| Theory | Scholar | Core Idea |
|--------|---------|-----------|
| **Behaviourist** | B.F. Skinner | Language = habit formed by imitation + reinforcement |
| **Nativist** | Noam Chomsky | Born with Universal Grammar (LAD) |
| **Interactionist** | Bruner, Vygotsky | Combination of innate ability + social interaction |
| **Social/Input theory** | Krashen | Comprehensible input (i+1) drives acquisition |

---

## Krashen's Monitor Model (Second Language Acquisition)

```
5 HYPOTHESES:
─────────────────────────────────────────────
1. ACQUISITION vs LEARNING
   Acquisition = subconscious (like children)
   Learning = conscious (studying grammar)

2. MONITOR HYPOTHESIS
   Learned rules act as an "editor" checking
   acquired language before output.
   
3. INPUT HYPOTHESIS (i+1)
   Best learning happens when input is just
   slightly beyond current level.
   i = current level, +1 = next step up.

4. NATURAL ORDER HYPOTHESIS
   Grammatical structures acquired in
   predictable order (not the order taught!)

5. AFFECTIVE FILTER HYPOTHESIS
   Anxiety, low motivation = high filter
   = less acquisition.
   Relaxed, motivated = low filter = more input
   gets in.
```

---

## Language Processing in the Brain

### Key Brain Areas

```
LEFT HEMISPHERE (for most right-handed people):
┌─────────────────────────────────────────┐
│  BROCA'S AREA (frontal lobe)           │
│  → Language PRODUCTION                  │
│  → Grammar and syntax                   │
│  Damage → Broca's aphasia:              │
│  "Speak... slow... no... grammar"       │
│  (telegraphic, effortful speech)        │
│                                         │
│  WERNICKE'S AREA (temporal lobe)        │
│  → Language COMPREHENSION               │
│  → Word meaning                         │
│  Damage → Wernicke's aphasia:           │
│  Fluent but meaningless speech,         │
│  can't understand others                │
└─────────────────────────────────────────┘
```

---

## Mental Lexicon

> The **mental lexicon** is the brain's internal dictionary — how words are stored and accessed in the mind.

- Words not stored alphabetically, but by **semantic networks** (meaning connections)
- **Priming effect**: Hearing "doctor" makes "nurse" easier to retrieve
- **Spreading activation**: When one word is activated, related words are also activated

---

## Language Production Models

```
LEVELT'S MODEL OF SPEECH PRODUCTION:
──────────────────────────────────────────
1. CONCEPTUALIZATION
   (What do I want to say?)
          │
2. FORMULATION
   (How to say it — choose words, grammar)
          │
3. ARTICULATION
   (Physically say it)
          │
4. SELF-MONITORING
   (Did I say it right? → feedback loop)
```

### Speech Errors (Evidence for mental structure)

| Error Type | Example | What it shows |
|-----------|---------|--------------|
| **Spoonerism** | "You have hissed all my mystery lectures" (for "missed all my history") | Sound planning happens before speaking |
| **Freudian slip** | Saying what you're really thinking | Thoughts affect production |
| **Word substitution** | "Pass the pepper" (wanting salt) | Semantic network errors |
| **Blend errors** | "Wug" (wanting to say "wasp" + "bug") | Words compete for selection |

# 11. Language Standardization, Pidgin & Creole

## Language Standardization

> **Standardization** is the process by which one dialect of a language is selected, codified, and promoted as the "correct" or "official" form.

### The Standardization Process (4 Stages)

```
1. SELECTION
   ─────────
   One variety chosen (usually the dialect
   of political/cultural power center)
   Example: London dialect → Standard English
        │
        ▼
2. CODIFICATION
   ──────────────
   Rules written down: grammars, dictionaries
   Example: Samuel Johnson's Dictionary (1755)
        │
        ▼
3. ELABORATION
   ─────────────
   Language expanded for all functions
   (science, law, literature, education)
        │
        ▼
4. ACCEPTANCE
   ──────────
   Community accepts this as the "standard"
   Taught in schools, used in media
```

### Types of Standards

| Type | Description | Example |
|------|-------------|---------|
| **Overt prestige** | High status, formal standard variety | BBC English, Standard American English |
| **Covert prestige** | Low status officially, but valued within a group | Local dialects, slang — makes you "cool" or "authentic" |

---

## Language Contact Situations

When speakers of different languages meet regularly (trade, colonization, migration), special contact languages emerge.

```
LANGUAGE CONTACT
       │
  ┌────┴────┐
  │         │
PIDGIN    BILINGUALISM
  │
  │ (if it survives and children learn it as L1)
  │
  ▼
CREOLE
```

---

## Pidgin

> **Definition:** A **pidgin** is a simplified contact language that develops when speakers of mutually unintelligible languages need to communicate (usually for trade or work). It is **nobody's first language**.

### Features of Pidgins:

| Feature | Description |
|---------|-------------|
| **Simplified grammar** | Reduced inflections, simpler verb forms |
| **Limited vocabulary** | Only what's needed for the contact situation |
| **No native speakers** | People use it alongside their own L1 |
| **Mixed lexicon** | Words from multiple languages |
| **Context-dependent** | Little used outside the contact situation |

> **Example:** Tok Pisin (Papua New Guinea), Nigerian Pidgin English, West African Pidgin

---

## Creole

> **Definition:** A **creole** is what happens when a pidgin becomes the **native language** of a new generation. It expands in vocabulary, grammar becomes more complex, and it becomes a full language.

### Pidgin → Creole Process:

```
GENERATION 1: Adults use pidgin for trade
              (No native speakers)
                     │
GENERATION 2: Children hear pidgin at home
              They acquire it as L1
              They EXPAND it (their brain fills gaps)
                     │
GENERATION 3: Full creole — complete language
              with native speakers
              Rich grammar, full vocabulary
```

### Key Differences

| Feature | Pidgin | Creole |
|---------|--------|--------|
| Native speakers | None | Yes |
| Grammar complexity | Simple | Full |
| Vocabulary | Limited | Expanding |
| Functions | Limited (trade, work) | All life functions |
| Examples | Early Tok Pisin | Haitian Creole, Jamaican Patois |

---

## Decreolization

> When a creole community has access to the prestige language it was based on, the creole may shift BACK toward that prestige language over time. This is **decreolization**.

```
CREOLE ←──── pressure from ───── PRESTIGE LANGUAGE
                                  (English, French, etc.)
```

Result: A **post-creole continuum**:
- **Basilect**: Furthest from prestige language (most creole-like)
- **Mesolect**: Middle ground
- **Acrolect**: Closest to prestige language

---

## Language Death and Endangerment

```
HEALTHY LANGUAGE
       │ (if no new speakers learn it)
       ▼
ENDANGERED LANGUAGE (few speakers, mostly elderly)
       │
       ▼
MORIBUND LANGUAGE (only handful of elderly speakers left)
       │
       ▼
EXTINCT LANGUAGE (last speaker dies — no more native speakers)
```

> **Example:** Latin → "died" as spoken L1 but "lives" in Spanish, French, Italian (its creole-like descendants!). Cornish (extinct 1777, now being revived).

## 10-Mark Question Formula (Suggestion from me)

For any 10-mark question, use this structure:

```
1. DEFINE the term (2 marks)
   "X is defined as..."

2. EXPLAIN with theory/scholars (2 marks)
   "According to [Scholar], ..."

3. GIVE examples (2 marks)
   "For example..."

4. SUBTYPES or RELATED CONCEPTS (2 marks)

5. REAL-WORLD APPLICATION / SIGNIFICANCE (2 marks)
```

## Final Exam Tips 🚨

1. **Always** write: Definition → Example → Elaboration
2. For consonants: **Voicing + Place + Manner** (e.g., voiced bilabial plosive = /b/)
3. For trees: **S → NP + VP** always comes first
4. Use **/ /** for phonemes, **[ ]** for phones — examiners check this!
5. Saussure's 3 big ideas: **Arbitrariness**, **Langue/Parole**, **Syntagmatic/Paradigmatic**
6. Chomsky's 3 big ideas: **UG/LAD**, **Competence/Performance**, **Deep/Surface Structure**
7. For pragmatics questions: Always bring in Grice or Speech Acts
8. Pidgin ≠ Creole: Pidgin has no native speakers, Creole does
9. When drawing IC trees, make sure every branch has exactly **2 immediate constituents**
10. **Context changes everything** — in semantics, pragmatics, AND sociolinguistics
